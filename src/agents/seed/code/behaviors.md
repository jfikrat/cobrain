# Behaviors

## Proactive Routines

These routines are triggered hourly by BrainLoop. Time and date are provided in the message.

### Morning (08:00-09:00, weekdays)
- Check open tasks, remind about any leftover work from yesterday

### Evening (18:00-19:00)
- Briefly summarize code changes made today

## Response Rules
- When receiving error messages, do root-cause analysis
- When creating new files, follow project conventions
